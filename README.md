## 👋 About Me

> **약 1년의 백엔드 개발 경험과 3년 6개월의 시스템 엔지니어 경험을 연결해, 운영 환경에서 신뢰할 수 있는 시스템을 만드는 개발자입니다.**

- 💻 **약 1년의 백엔드 개발 경험**을 통해 Python/FastAPI 기반 API, 주문 처리, 실시간 통신 시스템을 설계·운영했습니다.
- 🤖 실제 매장에서 약 8개월간 운영되고 글로벌 전시에 사용된 **바리스타 로봇 백엔드·제어 서버를 단독 담당**했습니다.
- 🖥️ **3년 6개월의 시스템 엔지니어 경험**을 바탕으로 서버 구축, 운영 자동화, 장애 대응 역량을 쌓았습니다.
- ⚡ `TCP`, `Socket.IO`, `ROS2 Rosbridge`, `Modbus TCP`를 활용해 로봇과 주변 장비를 실시간으로 연동했습니다.
- 🧩 기존 Flask 기반 구조를 `FastAPI / ASGI` 기반으로 전환하고 핵심 서비스의 책임을 분리했습니다.
- 🔍 개발에서 끝나지 않고 모니터링, 장애 복구, 운영 데이터 정합성까지 제품의 전체 생애주기를 다룹니다.
- 🎯 현재 **AI 서비스·로보틱스·Python 백엔드** 분야의 새로운 기회를 찾고 있습니다.
